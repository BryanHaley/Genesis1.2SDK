SDK Contents

WildTangent WildConvert WildConvert.exe
The WildTangent content creation tool. Use WildConvert to convert and compress JPGs, PNGs, WAVs, MIDs, and X files for use with the web driver.
JavaScript Template \JSTemplate\wtTemplate.html
Basic JavaScript Template - Covers the basics of using the WildTangent web driver in a web page.
JAVA Template \JAVATemplate\index.htm
Basic JAVA Template - Basic JAVA project using the web driver. This project is created using Microsoft Visual J++, however it will compile using any JAVA compiler.
Visual Basic Template \VBTemplate\VBTemplate.exe
Stand-Alone Visual Basic Template - Demonstrates how to use the web driver in a stand-alone Visual Basic program. This project was created using Microsoft Visual Basic 6.0.
www.wildtangent.com demos \demos\
Some sample demos from the www.wildtangent.com web site. For a complete listing of demo applications, please visit our web site.
